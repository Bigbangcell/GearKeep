# GearKeep 网页端功能增强设计文档

> **版本**：1.0
> **日期**：2026-04-24
> **状态**：已批准

## 1. 项目概述

本次更新旨在增强 GearKeep 网页端的用户体验和数据可视化能力，主要包括：
- 物品交互增强（悬停详情、点击跳转）
- 排序和筛选功能
- 自定义字段支持
- 数据看板和图表展示

## 2. 固定字段扩展

### 2.1 现有必填字段（保留）
- `name`: 物品名称（唯一标识，必填）
- `brand`: 品牌
- `model`: 型号
- `sn`: 序列号
- `purchasePrice`: 购入价格
- `purchaseDate`: 购入日期
- `warrantyMonths`: 保修月数
- `warrantyDeadline`: 保修截止日期
- `location`: 存放位置
- `notes`: 备注
- `isMaster`: 是否为主设备
- `createdAt`: 创建时间
- `updatedAt`: 更新时间

### 2.2 新增可选扩展字段
- `category`: 分类（相机、镜头、配件、其他等）
- `purchaseChannel`: 购买渠道（京东、天猫、官网、二手等）
- `condition`: 成色（全新、几乎全新、轻微使用、明显使用）
- `color`: 颜色

### 2.3 字段配置存储
在 IndexedDB 的 settings 存储中保存用户偏好：
```typescript
interface FieldSettings {
  visibleFields: string[];      // 可见字段列表
  fieldOrder: string[];         // 字段显示顺序
  listSortField: string;        // 当前排序字段
  listSortOrder: 'asc' | 'desc'; // 排序方向
}
```

## 3. 交互功能设计

### 3.1 物品卡片悬停详情

**卡片视图**：
- 鼠标悬停时显示半透明遮罩层
- 遮罩层显示物品所有字段的快速预览
- 详情气泡位于卡片上方或下方（自动判断边界）

**列表视图**：
- 鼠标悬停时整行高亮
- 右侧显示展开按钮，点击展开详情
- 或使用 tooltip 显示关键信息

**悬停详情内容**：
- 物品名称和品牌型号
- 价格（突出显示）
- 保修状态和剩余天数
- 存放位置
- 购买渠道和成色（如有）

### 3.2 排序功能

**实现方式**：
- 列表视图：点击表头文字触发排序
- 卡片视图：在工具栏添加排序下拉菜单
- 排序图标：升序 ↑ 降序 ↓ 无排序 —

**排序字段**：
- 名称（按拼音首字母）
- 价格（按数字大小）
- 购买日期（按时间）
- 保修截止（按时间）
- 品牌（按拼音首字母）
- 创建时间（按时间）

**排序逻辑**：
- 首次点击：升序排列（从小到大/从早到晚）
- 再次点击：降序排列（从大到小/从晚到早）
- 第三次点击：取消排序，恢复默认顺序

**中文拼音排序**：
- 使用 pinyin 库将中文转换为拼音首字母
- 按首字母顺序排序

### 3.3 字段显示/隐藏和排序

**设置位置**：
- 在器材库页面添加"字段设置"按钮
- 点击打开设置面板

**设置内容**：
- 可见字段复选框列表
- 字段拖拽排序
- 重置为默认按钮

**存储**：
- 保存到 IndexedDB settings
- 应用于首页和器材库

## 4. 数据看板设计

### 4.1 页面结构

**导航**：在导航栏添加"看板"链接

**页面布局**：
```
┌─────────────────────────────────────────────────┐
│  筛选栏：时间范围 | 分类 | 购买渠道              │
├─────────────────────────────────────────────────┤
│  统计卡片区                                      │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐     │
│  │总资产│ │日均 │ │最高  │ │最低  │ │物品数│     │
│  └─────┘ └─────┘ └─────┘ └─────┘ └─────┘     │
├─────────────────────────────────────────────────┤
│  图表区域（2列布局）                             │
│  ┌────────────────┐ ┌────────────────┐         │
│  │  分类占比饼状图  │ │  品牌占比饼状图  │         │
│  └────────────────┘ └────────────────┘         │
│  ┌────────────────┐ ┌────────────────┐         │
│  │  月度消费趋势图  │ │  价格区间分布图  │         │
│  └────────────────┘ └────────────────┘         │
│  ┌────────────────┐ ┌────────────────┐         │
│  │ 购买渠道分布图  │ │  保修状态分布   │         │
│  └────────────────┘ └────────────────┘         │
└─────────────────────────────────────────────────┘
```

### 4.2 统计卡片

| 卡片 | 计算方式 |
|-----|---------|
| 总资产 | 所有物品 purchasePrice 之和 |
| 日均消费 | 总资产 / 自首次购买以来的天数 |
| 最高价格 | MAX(purchasePrice) |
| 最低价格 | MIN(purchasePrice) |
| 物品总数 | 物品计数 |

### 4.3 图表类型

| 图表 | 类型 | 数据来源 |
|-----|------|---------|
| 分类占比 | 饼状图 | 按 category 分组统计 |
| 品牌占比 | 饼状图 | 按 brand 分组统计 |
| 月度消费趋势 | 折线图 | 按月份统计购买金额 |
| 价格区间分布 | 柱状图 | 按价格区间（0-1k, 1k-5k, 5k-10k, 10k+）统计 |
| 购买渠道分布 | 饼状图/柱状图 | 按 purchaseChannel 分组 |
| 保修状态分布 | 饼状图 | 保修中/即将到期/已过保 |

### 4.4 筛选功能

**筛选维度**：
- 时间范围：全部/最近一个月/最近三个月/最近一年/自定义
- 分类：全选/单选
- 购买渠道：全选/单选

**交互**：
- 筛选条件变更时，所有图表和数据自动更新
- 显示当前筛选条件的标签，可单独移除
